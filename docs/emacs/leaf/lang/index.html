日本語設定

2021-05-23
1 min read

日本語環境の設定を行います。

MozcというGoogleが公開しているIMEを使用します。 IMEはいろいろあるようですが、Mozcが多く使われている印象。

emacs-mozcをインストール

$ sudo apt install emacs-mozc-bin

init.elに以下の設定を追加

(leaf *language-settings
  :config
  (set-language-environment 'Japanese)
  (prefer-coding-system 'utf-8)
  (leaf mozc
    :ensure t
    :config
    (setq default-input-method "japanese-mozc")
    (global-set-key (kbd "C-o") 'toggle-input-method)
    )  
  )

leafにはパッケージの設定をまとめることができ、 leafをまとめるためだけのleafブロックの名前は アスタリスク*を使用することが推奨されています。 ([正式リリース]leaf.elで雑然としたEmacs設定ファイル「init.el」をクリーンにするより) そのため言語設定はまとめて*language-settingsとします。

:configというキーワードはleafブロック内で最後に展開され、パッケージそのものに関係ない周辺設定などを書きます。
