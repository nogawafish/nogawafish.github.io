pandas_datareader で株・為替・仮想通貨情報を取得

2021-06-05
2 min read

pandas_datareaderというライブラリを使えば株価・為替・暗号通貨等のデータを 簡単に取得できます。

pandas_datareaderは実態はスクレイピングを代行してくれるライブラリです。

インストール

仮想環境を起動

(pyvenv-workon "nogawafishSite-gnCTSypq")

インストール

pip install pandas-datareader

バージョン確認

import pandas_datareader
pandas_datareader.__version__
0.9.0

取得できるデータ

pandas-datareaderのdocumentによるとTiingo, IEX, Alpha Vantage …, Naver Finance といろいろなサイトのデータが取れるようです。 サイト数が多くすべては把握しきれないですが、株・為替・暗号通貨の価格が得られることを検証しました。

株

日経225

import datetime
import pandas_datareader.data as web

start = datetime.date(2015,1,1)
end = datetime.date.today()

df = web.DataReader('^N225', 'yahoo', start, end)

print(df.head())
print(df.tail())
