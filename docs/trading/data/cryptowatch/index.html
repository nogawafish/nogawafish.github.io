Cryptowatchによる暗号通貨データ取得

2021-06-13
2 min read

pandas-datareaderでは暗号通貨の分足・時足が取れないぽいのですが、 仮想通貨の分足・時足データを取るにはCryptowatchというサイトのapiを使えば 取得できることが分かりました。

Cryptowatchとは

Cryptowatch is a real-time crypto markets platform. We connect you to the global crypto markets with tools that enable research, analysis, monitoring, and order execution.

とAPI referenceでは言っています。

見たところ興味のあるデータとしては OHLC Candlsticks, Order Book, Trades(約定履歴)などが取得できるようです。

OHLC取得確認

bitflyer, btcjpyの現時刻以降の日足OHLC(Open High Low Close)を得る例を実行します。 データ量が多くても見にくいので現時刻以降のデータを取ります。

仮想環境を起動

(pyvenv-workon "nogawafishSite-gnCTSypq")

requestsライブラリでAPIを叩く

import requests
import time
from datetime import datetime
from pprint import pprint
now_unix_time = int(time.time())
# リクエスト実行
response = requests.get(
    "https://api.cryptowat.ch/markets/bitflyer/btcjpy/"
    f"ohlc?periods=86400&after={now_unix_time}"
)
# 時刻確認
print(f"unix time:{now_unix_time}")
print(f"jst      :{datetime.fromtimestamp(now_unix_time)}")
unix time:1623584255
jst      :2021-06-13 20:37:35

このリクエスト結果を確認します。

pprint(response.json(), sort_dicts=False)
{'result': {'86400': # 86400s = 1day 
                      [[ 
                       1623628800, # CloseTime 
                       3904362,    # Open
                       3970000,    # High
                       3824501,    # Low
                       3943614,    # Close
                       1648.19183709, # Volume
                       6415397989.940934 # QuoteVolume
		       ]]},
 'allowance': {'cost': 0.015,
               'remaining': 9.91,
               'upgrade': 'For unlimited API access, create an account at '
                          'https://cryptowat.ch'}}

と上のようなjson結果が得られます。

ちなみにclose timeを標準時間に変換していつのことか確認します。

print(f"jst      :{datetime.fromtimestamp(1623628800)}")
jst      :2021-06-14 09:00:00

現時刻(6/13 20:37)以降のbitflyer/btcjpyの日足をリクエストし、 Close time:1623628800(6/14 9:00)の日足が得られました。 最初バグかと思いましたが、そうではなく日足のクローズが9:00で、 現時刻までの価格で暫定的なohclを出してくれているのだと思います。

コストに注意

登録なしユーザーはリクエスト量に上限があります。 'allowance'にその情報が送られてくるようです。 'cost'は一回のリクエストにかかるコストで、 登録なしユーザーは'remaining'が切れないようにリクエスト回数を注意する必要があります。 この'allowance'は24時間ごとにリセットされるそうです。

他のデータを取得する場合

"https://api.cryptowat.ch/markets/bitflyer/btcjpy/ohlc?periods=86400&after=1623628800" の条件をいろいろ変えることでいろんなデータが得られるようです。

詳細はAPI referenceに詳しいのでそちらを参照ください。
